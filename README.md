# World-Population-Dashboard
This interactive dashboard provides a comprehensive view of global population trends, enabling users to explore demographic insights across countries, regions, and continents. Built in Power BI, it combines clear visualizations with dynamic filters for deeper analysis.
# Key Metrics (KPI Cards)
• Total Population:96.94M  
• Male Population:48.79M  
• Female Population:48.14M  
• Highlighted Age Group:0–4 years  

# Visualizations
• Line Chart:Population trends showing growth in total and male population.  
• Pie Chart:Gender distribution (Female:57.41%, Male: 42.59%).  
• Bar Chart:Population by age groups (0–4, 5–9, 10–14, etc.).  
• Map Visualization:World map displaying population distribution across continents.  
• Table:Country-level breakdown (e.g:Afghanistan by age group and gender).  

# Insights
• Compare male vs female population across age groups.  
• Track global population growth trends over time.  
• Identify the most populous age groups worldwide.  
• Explore country-specific demographics with detailed tables.
